Esse sistema foi desenvolvido pelo grupo acima como tarefa da disciplina 'Programação OO' ministrada pelo professor Maurício Amorim no IFF Fluminense.

Com o desenvolvimento desse sistema foi possível fixar os conhecimentos aprendidos na aula, sobre o Spring e a arquitetura MVC. Além de exigir conhecimentos em JPA para a realização dos relacionamentos entre as classes e também de conhecimentos gerais em linguagens/frameworks web como Bootstrap no desenvolvimento do layout das páginas.
